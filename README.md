<h1 align="center">🌡️ Temperature Converter (Celsius ↔ Fahrenheit)</h1>

<p align="center">
  A simple and efficient temperature converter that allows users to convert values between 
  <b>Celsius</b> and <b>Fahrenheit</b> instantly.
</p>

<h2>🚀 Features</h2>

  <li>🔄 Convert Celsius to Fahrenheit</li>

  <li>💡 Beginner-friendly project</li>
  <li>📱 Lightweight and easy to use</li>
</ul>

<h2>📐 Conversion Formula</h2>

<p><b>Celsius to Fahrenheit:</b></p>
<pre>F = (C × 9/5) + 32</pre>

<p><b>Fahrenheit to Celsius:</b></p>
<pre>C = (F - 32) × 5/9</pre>
